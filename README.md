
# 🔐 Secure Authentication System

## Overview
As part of my cybersecurity learning journey, I developed this Secure Authentication System in Python to explore how user authentication works in real-world applications.
 

This project allows users to register, log in securely, and records authentication events for monitoring purposes.

---

## Features

- User registration
- Password hashing using SHA-256
- User login authentication
- Failed login attempt tracking
- Security event logging
- Persistent user storage using JSON

---

## Cybersecurity Concepts Demonstrated in the project

- Authentication
- Password hashing
- Brute-force protection
- Security logging
- Secure credential storage
